# Task Manager

Task Manager — это мобильное приложение, разработанное на **React Native**, которое позволяет эффективно управлять задачами.

## 🎥 Видео демонстрация работы приложения

[Посмотреть видео работы приложения с Google Drive](https://drive.google.com/file/d/1FhUJ4e3vofomS3MbIp_TX3VoXszeyUL1/view?usp=sharing)  

## 📱 Основной функционал

- **Добавление задач** — легко создавайте новые задачи с указанием всех необходимых деталей.
- **Изменение задач** — редактируйте существующие задачи в любое время.
- **Удаление задач** — удаляйте ненужные задачи в любое время.
- **Просмотр истории** — доступ к истории работы с задачми.
- **Оповещения** — напоминания за **30 минут до окончания задачи**.

## 🗺️ Интеграция с картами и геолокацией

- **Встроенная карта** для визуализации местоположений.
- Использование **Yandex Geocoder API** для получения и отображения адресов на основе координат.

## 📂 Работа с файлами

- Поддержка **работы с хранилищем файлов**, чтобы можно было загружать и сохранять данные прямо в приложении.

## 🎨 Темизация

- Поддержка **светлой и тёмной темы** для комфортного использования в любое время суток.
